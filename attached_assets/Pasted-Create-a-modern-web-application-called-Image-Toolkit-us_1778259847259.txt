Create a modern web application called "Image Toolkit" using Next.js, TypeScript, Tailwind CSS, and React.

The application must run entirely on the client side without requiring a backend server or image uploads. All image processing should happen locally in the user's browser for privacy and performance.

Main Features:

* Drag and drop image upload area
* Support selecting multiple images
* Image preview grid
* Resize images by width and/or height
* Maintain aspect ratio toggle
* Compress image quality with a slider
* Convert between PNG, JPG, and WEBP formats
* Download processed images individually
* Download all processed images as a ZIP file
* Remove image metadata when exporting
* Show original vs processed file size
* Dark mode support
* Responsive mobile-friendly UI
* Smooth modern animations and transitions
* Clean minimal design similar to modern SaaS tools

Technical Requirements:

* Use Next.js App Router
* Use TypeScript everywhere
* Use Tailwind CSS for styling
* Use react-dropzone for drag and drop uploads
* Use pica for high-quality image resizing
* Use JSZip for ZIP downloads
* Use browser APIs only (Canvas API, File API, Blob API)
* No backend or database
* No authentication
* Fully static deploy compatible

UI Requirements:

* Beautiful landing page
* Upload area with dashed border and hover effects
* Image cards with preview thumbnails
* Controls panel for resize/compression options
* Progress indicators during processing
* Download buttons with modern styling
* Minimalistic and professional aesthetic
* Rounded corners and soft shadows
* Accessible keyboard navigation

Extra Features:

* Drag to reorder images
* Batch processing
* Compare before and after image sizes
* Optional crop tool
* PWA support for offline usage
* Fast performance even with large images

Project Structure:

* Organized reusable components
* Clean folder architecture
* Reusable hooks for image processing
* Utility functions separated properly
* Strong typing and clean code practices

The generated code should be production-ready, scalable, and easy to maintain.
